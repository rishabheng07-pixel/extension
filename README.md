# Tracky 🔥

**Track your daily coding time with a streak system. Automatic idle detection, persistent stats, and achievement badges.**

A lightweight VS Code extension that motivates you to code consistently by tracking your coding time and maintaining daily streaks.

## Features

### 🎯 Core Features

- **Automatic Time Tracking**: Starts tracking automatically when VS Code is open
- **Smart Idle Detection**: Automatically pauses tracking when you're inactive (60 seconds of no window focus)
- **Daily Streak System**: Build a streak by coding at least 10 minutes per day
  - Streak increments each day you meet the minimum
  - Streak resets if you skip a day
- **Persistent Stats**: All data saved to VS Code's global state (syncs across all workspaces)
- **Real-time Status Bar**: See your stats at a glance in the VS Code status bar

### 📊 Tracked Statistics

- **Today's Coding Time**: How long you've coded today
- **Total Coding Time**: Cumulative time across all days
- **Current Streak**: Consecutive days with ≥10 minutes of coding
- **Achievement Badges**: Special badges for milestones:
  - 🏆 **7-Day Warrior** - Reach a 7-day streak
  - ⚡ **10-Hour Champion** - Code for 10+ hours total
  - 👑 **Century Coder** - Code for 100+ hours total
